# 🚨 LandSpy Guard

## 📌 Overview
LandSpy Guard is an IoT-based landmine detection robot using ESP32-CAM, GPS, and sensors.

## 🔥 Features
- 📍 Real-time GPS location tracking
- 📷 Live video streaming (ESP32-CAM)
- ⚠️ Landmine detection alert
- 🌐 Web-based control system
- 🤖 Remote robot navigation

## 🛠️ Components Used
- ESP32-CAM
- GPS Neo-7M
- Ultrasonic / Metal sensor
- Motor Driver (L298N)
- WiFi Module (inbuilt)

## ⚙️ Working
1. Robot scans area
2. Sensor detects landmine
3. GPS sends location
4. Camera shows live video
5. Data displayed on web page

## 📡 Future Scope
- AI-based detection
- GPR integration
- Cloud data storage
